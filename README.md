# myrime
我的 rime 配置文件

rime_ice.custom.yaml---雾凇拼音的自定义文件

double_pinyin.custom.yaml---自然码双拼的自定义文件

weasel.custom.yaml---Windwos 平台的主题自定义文件

my.trime.yaml---安卓平台的 trime 同文输入法的外观主题，可以把每页显示候选词数量设定为 50 或 99，来更好地配合这个主题
